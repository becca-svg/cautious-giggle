FTP

Definition: FTP (File Transfer Protocol) is a standard network protocol used for the transfer of computer files between a client and server on a computer network.

FTP works by establishing two connections between the client and server: a control connection for sending commands and a data connection for transferring files. It uses a username and password for authentication.

FTP is commonly used for uploading and downloading files from web servers, as well as for sharing files within a local network.
