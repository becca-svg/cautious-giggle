Web 1.0

Definition: Web 1.0 refers to the early stage of the World Wide Web, characterized by static web pages and limited user interaction.

During the Web 1.0 era, websites were primarily used to display information and had little to no user-generated content. Users were passive consumers of web content, and websites were often simple HTML pages with basic styling.

Web 1.0 laid the foundation for the modern web, introducing concepts like URLs, browsers, and the basic infrastructure needed for web communication.
