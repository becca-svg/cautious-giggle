World Wide Web (WWW)

Definition: The World Wide Web is a system of interlinked web pages accessed via the internet.

People often confuse the Web with the internet. The internet is the network, while the Web is the information (websites, pages, links) you access on it.

Tim Berners-Lee invented the WWW in 1989. He also created the first web browser and web server.
